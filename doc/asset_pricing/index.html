Fixed Effect Estimation¶

class LinearFactorModel(portfolios, factors, *, constant=True, sigma=None)[source]¶

Asset pricing linear factor models estimator

Parameters:
  • portfolios (array-like) – nobs by nportfolio array of test portfolios
  • factors (array-like) – nobs by nfactor array of priced factors
  • constant (bool, optional) – Flag indicating whether to include a constant when estimating portfolio loadings
  • sigma (array-like, optional) – nportfolio by nportfolio positive definite covariance matrix of portfolio residuals for use in estimation. If not provided, an identify matrix is used

Notes

Implements both time-series and cross-section estimators of risk premia, factor loadings and model tests

Parameters:method ({'cs', 'ts', 'time-series', 'cross-section'}) – Method to use when estimating risk premia and constructing model tests
Returns:results – Results class with parameter estimates, covariance and test statistics
