Linear Asset Pricing Models¶

Estimation¶

class TradedFactorModel(portfolios, factors)[source]¶

Asset pricing linear factor models estimator for traded factors

Parameters:
  • portfolios (array-like) – array of test portfolio returns (nobs by nportfolio)
  • factors (array-like) – array of priced factor returns (nobs by nfactor

Notes

Implements both time-series estimators of risk premia, factor loadings and zero-alpha tests.

The model estimated is

\[r_{p,it}^e = \alpha_i + f_t \beta_i + \epsilon_{it}\]

where \(r_{p,it}^e\) is the excess return to test portfolio i. The model is directly tested using the estimated values :math:hat{alpha}_i. Risk premia, \(\lambda_i\) are estimated using the sample averages of the factors, which must be excess returns on traded portfolios.

fit(cov_type='robust', debiased=True, **cov_config)[source]¶
Parameters:
  • cov_type ({'robust', 'kernel'}) – Name of covariance estimator
  • debiased (bool, optional) – Flag indicating whether to debias the covariance estimator using a degree of freedom adjustment
  • **cov_config (dict) – Optional keyword arguments to pass to covariance estimator
Returns:

results – Results class with parameter estimates, covariance and test statistics
Return type:

TradedFactorModelResults

Notes

The kernel covariance estimator takes the optional arguments kernel, one of ‘bartlett’, ‘parzen’ or ‘qs’ (quadratic spectral) and bandwidth (a positive integer).
