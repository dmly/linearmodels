Formulas and Mathematical Detail¶

There are \(N\) test portfolios, \(r_{t}^{e}\) (\(p\) by 1) which are assumed to be excess returns (zero-investment portfolios) and \(K\) factors, \(f_{t}\) (\(k\) by 1). For each there are \(T\) observations.

Linear Factor Model for Traded Factors¶

This model is a Seemingly Unrelated Regression (SUR). All models share the same regressors so that the model for test portfolio \(i\) is

\[r_{it}^{e}=\alpha_{i}+\beta_{i}f_{t}+\epsilon_{it}.\]

Inference is based on the sandwich covariance estimator

\[\hat{\Sigma}=\left(T-df\right)^{-1}A^{-1}BA^{-1}\]

where

\[\hat{A}=I_{N}\otimes\Sigma_{\left[1\:f^{\prime}\right]}\]

and

\[\Sigma_{\left[1\:f^{\prime}\right]}=T^{-1}\sum_{t=1}^{T}\tilde{f}_{t}\tilde{f}_{t}^{\prime}\]

where \(\tilde{f}_{t}=\left(1\:f^{\prime}\right)^{\prime}\) is the set of factor returns augmented with a constant. Define the stacked vector of scores

\[s_{t}=\left[\epsilon_{t}\otimes\tilde{f}_{t}^{\prime}\right].\]

If the heteroskedasticity robust estimator is used,

\[\hat{B}=T^{-1}\sum_{t=1}^{T}s_{t}s_{t}^{\prime}.\]

If the HAC estiamtor is used,

\[\begin{split}\begin{aligned} \hat{B} & =T^{-1}\left\{ \hat{\Gamma}_{0}+\sum_{i=1}^{bw}K\left(i,bw\right)\left(\hat{\Gamma}_{i}+\hat{\Gamma}_{i}^{\prime}\right)\right\} \\ \hat{\Gamma}_{j} & =\sum_{t=j+1}^{T}s_{t}s_{t-j}^{\prime}\end{aligned}\end{split}\]

where \(K\left(i,bw\right)\) is a kernel weighting function and \(bw\) is the bandwidth.

The J statistic¶

The J statistic is implemented as

\[J=\hat{\alpha}^{\prime}\hat{\Sigma}_{\alpha}^{-1}\hat{\alpha}^{\prime}\sim\chi_{N}^{2}\]

where \(\hat{\Sigma}_{\alpha}\) consists of rows and columns\(1,(k+1)+1,2(k+1)+1,\ldots\) of \(\hat{\Sigma}\).

Linear Factor Model for Non-traded Factors¶

When factors are not traded, the SUR approach cannot be used since the expected value of the factor is not equal to its risk premium. One solution is to use a 2-step estimator. The first step regresses the test portfolio returns on the factors and a (throw-away) constant. This constant is included only to demean the factors to compute the correct factor loadings. The second stage then uses the estimated factor loadings and the average portfolio returns to estimate the risk premia.

The first stage regressions are virtually identical to the SUR regressions except the constant is no longer \(\alpha_{i}\),

\[r_{it}^{e}=c_{i}+\beta_{i}f_{t}+\epsilon_{it}.\]

The second stage regression then uses the estimated coefficients as regressors to estimate the risk premia,

\[\bar{r}_{i}^{e}=\left[1_{N}\,\hat{\beta}\right]\lambda+\eta_{i}.\]

Note that the residual from this regression\(\hat{\eta}_{i}=\hat{\alpha}_{i}\), the excess compensation test portfolio \(i\) is generating. If the risk free rate is assumed to be 0, then the column of 1s is not included and \(\lambda\) is \(K\) by 1.

Inference is conducted by treating this as a 3-step GMM problem where the first two steps were jsut described and the third estimated the model \(\hat{\alpha}\). The moment conditions corresponding to the FOC of these three steps are

\[\begin{split}g_{t}=\left[\begin{array}{c} \epsilon_{t}\otimes\left[1\,f_{t}^{\prime}\right]^{\prime}\\ \beta^{\prime}\left(r_{t}-\left[1_{N}\,\beta\right]\lambda\right)\\ r_{t}-\left[1_{N}\,\beta\right]\lambda-\alpha \end{array}\right]\end{split}\]

where \(\epsilon_{it}=r_{it}^{e}-c_{i}-\beta_{i}f_{t}\), \(\beta\) is \(N\) by \(K\), \(f_{t}\) is \(K\) by 1, \(\lambda\) is \(K\) by 1, \(r_{t}\) is \(N\) by 1, \(\epsilon_{t}\) is \(N\) by 1 and\(\alpha\) is \(N\) by 1. Inference is based on

\[\hat{\Sigma}=T^{-1}G^{-1}S\left(G^{-1}\right)^{\prime}\]

where the Jacobian \(G\) is estimated by

\[\begin{split}\hat{G}=-\left[\begin{array}{ccc} I_{N}\otimes\Sigma_{\left[1\,f\right]} & 0 & 0\\ G_{21} & \left[1_{N}\,\beta\right]^{\prime}\left[1_{N}\,\beta\right] & 0\\ I_{N}\otimes\left[0\ \lambda\right] & \left[1_{N}\,\beta\right] & I_{N} \end{array}\right]\end{split}\]

The block \(G_{21}\)is

\[G_{21}=\left[G_{21,1}\ G_{21,2}\:\ldots\ G_{21,N}\right]\]
\[G_{21,i}=\left[0_{\left(K+1\right)\times1}\;\left(\left[1\;\beta_{i}\right]^{\prime}\lambda^{\prime}-\left[0_{K}\;E_{T}\left[u_{it}\right]I_{K}\right]^{\prime}\right)\right]\]

where expectations are replaced by sample averaged. When the risk free rate is 0, all terms of the form \([1_{N}\;\beta]\) become \(\beta\) and

\[G_{21,i}=\left[0_{K\times1}\;\left(\beta_{i}^{\prime}\lambda^{\prime}-E_{T}\left[u_{it}\right]I_{K}\right)\right]\]

\(S\) is an estimator of the scores and its form is identical to either the heteroskedasticity consistent estimator or the HAC estimator in the previous section.

The J statistic¶

The J statistic is virtually identical except that it has a difference distribution.

\[J=\hat{\alpha}^{\prime}\hat{\Sigma}_{\alpha}^{-1}\hat{\alpha}^{\prime}\sim\chi_{N-K}^{2}\]

The loss of degrees of freedom is obvious since \(\hat{\alpha}=\hat{\eta}\) which are regression residuals from a regression of \(N\) average returns on \(K\) factor loadings. \(\hat{\Sigma}_{\alpha}\) is the bottom right block of \(\hat{\Sigma}\).

GMM Estimation of Linear Factor Models¶

The GMM estimator solves a set of non-linear moment conditions of the form

\[\begin{split}\left[\begin{array}{c} \epsilon_{t}\otimes\left[1\:f_{t}^{\prime}\right]^{\prime}\\ f_{t}-\mu \end{array}\right]\end{split}\]

where

\[\epsilon_{t}=r_{t}^{e}-\left[1_{N}\;\beta\right]\lambda-\beta\left(f_{t}-\mu\right)\]

is the vector of pricing errors. The factors can be traded or non-traded. This assumes that the model for expected returns is

\[E\left[r_{it}^{e}\right]=\lambda_{0}+\sum_{i=1}^{k}\beta_{i}\lambda_{i}\]

so that the risk-free rate might not be zero. This form is sometimes used with excess returns if there is doubt about the risk-free rate used. The Jacobian is then

\[\begin{split}-\left[\begin{array}{ccc} I_{N}\otimes\left(\tilde{\lambda}+f_{t}-\mu\right)\left[1\:f_{t}^{\prime}\right]^{\prime} & \left[\begin{array}{c} \left[1\:f_{t}^{\prime}\right]^{\prime}\left[1\!\beta_{1}\right]\\ \vdots\\ \left[1\:f_{t}^{\prime}\right]^{\prime}\left[1\!\beta_{N}\right] \end{array}\right] & \left[\begin{array}{c} -\left[1\:f_{t}^{\prime}\right]^{\prime}\beta_{1}\\ \vdots\\ -\left[1\:f_{t}^{\prime}\right]^{\prime}\beta_{N} \end{array}\right]\\ 0 & 0 & I_{K} \end{array}\right]\end{split}\]

When the risk-free rate is not estimated (assumed to be 0), then the residuals are

\[\epsilon_{t}=r_{t}-\beta\lambda-\beta\left(f_{t}-\mu\right)\]

and the Jacobian is

\[\begin{split}-\left[\begin{array}{ccc} I_{N}\otimes\left(\lambda+f_{t}-\mu\right)f_{t} & \left[\begin{array}{c} \left[1\:f_{t}^{\prime}\right]^{\prime}\beta_{1}\\ \vdots\\ \left[1\:f_{t}^{\prime}\right]^{\prime}\beta_{N} \end{array}\right] & \left[\begin{array}{c} -\left[1\:f_{t}^{\prime}\right]^{\prime}\beta_{1}\\ \vdots\\ -\left[1\:f_{t}^{\prime}\right]^{\prime}\beta_{N} \end{array}\right]\\ 0 & 0 & I_{K} \end{array}\right].\end{split}\]
